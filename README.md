# Urban Dictionary RESTful API

## Description

A simple API to get data from Urban Dictionary

## Documentation

You can check the endpoints in [Swagger](https://urbandictionaryapi.herokuapp.com)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
